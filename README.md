# -Prototype-navigation-autonome
Système de sécurité pour navigation autonome sur microcontrôleur Arduino (C++) avec évitement d'obstacles et arrêt d'urgence. Intégration des capteurs HC-SR04 et PIR, pilotage de servo-moteur par filtrage de données. Développement d'un système embarqué autonome avec interface HMI (Buzzer/LED) pour la gestion des alertes de proximité (2 cm à 4 m).
